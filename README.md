# DigiCanji v0.3.5

## 更新内容

- なぞり書きの模範字を明朝体から教科書向けゴシック系へ変更
- 模範解答をパッドの縦横比に合わせ、正方形に近い文字領域へ最大表示
- 複数文字は、1行に圧縮せず複数行・複数列へ自動配置
- 文字の縦横方向の変形を廃止
- なぞり書きの認識欄を実動化
- 認識候補と模範解答の一致・不一致を表示
- 通常問題となぞり書き双方で、認識用の筆跡点を適度に間引いて送信量を削減
- 古い認識要求を中止し、最新の筆跡だけを評価
- 成功した認識接続先を次回から優先
- 同一筆跡の認識結果をキャッシュ

`DigiCanji_Editor.html` が生成する単一HTMLにも、同じプレイヤー v0.3.5 が内蔵されています。
