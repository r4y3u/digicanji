# DigiCanji v0.2.0

漢字の読みを見て、手書きで適切な表記を答える学習アプリです。配布用の `DigiCanji.html` には問題データ・UI・手書き認識処理をすべて同梱しています。

## 起動

`DigiCanji.html` をブラウザで開きます。GitHub Pagesへ置く場合も、この1ファイルだけで起動できます。

手書き認識は、対応ブラウザでは Handwriting Recognition API を試し、利用できない場合はオンライン認識へフォールバックします。フォールバック利用時はネットワーク接続が必要です。

## 問題データの位置付け

実行中のCSV読込・貼り付け機能はありません。問題データは、HTML内の次の要素へJSONとして埋め込まれています。

```html
<script id="digicanji-quiz-data" type="application/json">
{
  "quizId": "...",
  "title": "...",
  "questions": [
    {
      "front": "改善を",
      "prompt": "ハカル",
      "back": "。",
      "answer": "図る。",
      "explanation": "しようと考えること"
    }
  ]
}
</script>
```

将来のエディターは、編集用プロジェクトやCSVからこのJSONを生成し、完成した `DigiCanji.html` へ埋め込む構成を想定しています。`quizId` ごとに累計出題回数と正解数が分離されます。

解答末尾と後部冒頭の句読点が重なる場合、その句読点は手書き対象から自動除外されます。

## 操作

### 横書き

- 認識枠クリック：文字枠を選択
- `←` / `→`：文字枠を移動
- `↑`：見直しメニュー
- `↓`：進む／回答
- `1`～`9`、`0`：1～10問目へ移動
- `Z`：1画消す

### 縦書き・ワイド／ポータブル

- 認識枠クリック：文字枠を選択
- `↑` / `↓`：文字枠を移動
- 右利き：`←`で進む／回答、`→`で見直し
- 左利き：`→`で進む／回答、`←`で見直し
- `1`～`9`、`0`：1～10問目へ移動
- `Z`：1画消す

縦書き・ポータブルは、スマートフォンの縦長画面内で論理上のワイド画面全体を90度回転します。端末を横に持つと、問題・手書きパッド・操作盤の3領域が左右に並んで見えます。画面自体が横長の場合は、回転せずワイド表示になります。

## 主な仕様

- 最大10問、重複なし
- 正答率が低い問題・出題回数が少ない問題を優先した重み付きランダム
- 問題・文字枠ごとの筆跡と認識文字を一時保存
- 解答文字数に応じて認識欄を分割
- 右利き／左利きの配置反転
- 横書きでは手書きパッドを正方形化
- 縦書きでは認識枠を問題文の左右に縦配置
- 縦書きの見直しメニューと操作ボタンを縦組み化
- 最終問題の「回答」を強調表示
- 縦書きの答え合わせは1問目を右端として右から左へ配置
- 未回答は文字を表示せず、傍線だけを表示
- 結果文クリックで解説表示
