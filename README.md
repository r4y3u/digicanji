# DigiCanji v0.3.2

手書き漢字認識を使った学習用クイズアプリと、問題データを編集して配布用HTMLを生成するエディターです。

## 収録ファイル

- `DigiCanji.html`
  - テスト問題を同梱した単一HTML版プレイヤー
  - CSS・JavaScript・問題データを1ファイルに内包
- `DigiCanji_Editor.html`
  - DigiCanji専用エディター
  - 最大1000問の編集、CSV入出力、編集用プロジェクト保存、単一HTML生成に対応
- `source/`
  - プレイヤーの分割ソース
- `THIRD_PARTY_NOTICES.md`
  - 第三者由来データ等の表記

## v0.3.2の変更点

### なぞり書き復習

- リザルトの各問題を選択して開く解説ポップアップへ、`いま書く` ボタンを追加
- `いま書く` を選ぶと、対象問題専用のなぞり書きウィンドウを表示
- 問題文と「フリー」認識枠を表示
- 手書きパッド内へ模範解答を薄く表示
- 模範解答の上から自由になぞり書き可能
- `1画消す`、`クリア`、`OK` に対応
- `OK` でなぞり書きを終了し、元のリザルト画面へ復帰
- 横書き、縦書き・ワイド、縦書き・ポータブル、左右の利き手設定へ対応
- ポータブル回転時も、ブラウザ差を抑えるローカル座標優先方式で筆跡位置を算出

### エディター v0.2.1

- 生成するプレイヤーをv0.3.2へ更新
- エディターから生成した単一HTMLにも、なぞり書き復習を同梱

## エディターの入力検査

各問題について、次の条件を確認します。

- 問部が入力されている
- 解答が入力されている
- 問部と解答が同じではない
- 解答が、ひらがなと漢字だけで構成されている

解答がひらがなのみの場合は、読みを問う問題として扱い、プレイヤーの認識枠を `フリー` にします。

```csv
前部,問部,後部,解答,解説
改善を,ハカル,。,図る,しようと考えること
,アツイ,本を読む。,厚い,物を横から見たときの幅がある
「厚い」の読みを,答えなさい,。,あつい,「厚い」は「あつい」と読む。
```

解答欄には句読点、空白、数字、記号、カタカナを入力できません。句読点は後部へ入力してください。

## 基本ワークフロー

1. `DigiCanji_Editor.html` をブラウザで開く
2. 表へ問題を入力、またはCSVを読み込む
3. エラー表示がないことを確認する
4. タイトルとクイズIDを設定する
5. `HTML生成` を押す
6. 生成されたHTMLを配布またはGitHub Pagesへ配置する

プレイヤーにはCSV読込機能を設けていません。問題データは生成時にHTMLへ同梱されます。

## キーボード操作（プレイヤー）

### 横書き

- `← / →`：文字枠移動
- `↑`：見直し
- `↓`：進む／回答

### 縦書き

- `↑ / ↓`：文字枠移動
- 右利き：`←` が進む／回答、`→` が見直し
- 左利き：`→` が進む／回答、`←` が見直し

### 共通

- `1～9 / 0`：1～10問目へ移動
- `Z`：1画消す
- `Esc`：メニュー、解説、なぞり書きを閉じる
- なぞり書き中の `Enter`：`OK`

フリー認識問題では文字枠移動は行いません。

## 確認項目

- JavaScript構文検査
- 横書きのなぞり書き表示、描画、1画消し、クリア、OK
- 縦書き・ワイドのなぞり書きレイアウト
- 縦書き・ポータブルの回転レイアウト
- 模範解答がユーザー筆跡より薄く表示されること
- OK後にリザルト画面へ戻ること
- エディターのHTML生成テンプレートへ新プレイヤーが内蔵されていること
- プレイヤーとエディターで実行時エラーが発生しないこと
