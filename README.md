# DigiCanji v0.3.0

手書き漢字認識を使った学習用クイズアプリと、問題データを編集して配布用HTMLを生成するエディターです。

## 収録ファイル

- `DigiCanji.html`
  - テスト問題を同梱した単一HTML版プレイヤー
  - CSS・JavaScript・問題データを1ファイルに内包
- `DigiCanji_Editor.html`
  - DigiCanji専用エディター初期版
  - 問題編集、CSV入出力、編集用プロジェクト保存、単一HTML生成に対応
- `source/`
  - プレイヤーの分割ソース
- `THIRD_PARTY_NOTICES.md`
  - 第三者由来データ等の表記

## v0.3.0の変更点

### プレイヤー

- 縦書き・ポータブルで、CSS回転されたキャンバスのタッチ座標取得を修正
  - キャンバス固有のローカル座標を優先
  - 利用できない環境では従来の矩形座標変換へフォールバック
  - iOS版Braveなど、指より上に筆跡が残る現象を抑制
- 縦書きの見直し一覧を整理
  - 番号を上、入力状態を下、問題文を中央へ配置
  - 問題文は通常1列、長い場合のみ複数列へ折り返し
  - `入力済` を `入力済み` に変更
- 縦書きの答え合わせを整理
  - 問題番号後の句点を削除
  - 番号・問題文・正誤判定の間隔と区切りを調整
  - 長い問題文のみ複数列表示
  - 未回答は文字を表示せず、傍線だけを表示

### エディター v0.1.0

- 100行の表形式編集
- 列：`前部 / 問部 / 後部 / 解答 / 解説`
- ExcelやGoogleスプレッドシートからの複数セル貼り付け
- 必須項目チェック（問部・解答）
- 選択行に連動するプレビュー
- CSV読込・保存
- 編集用 `.digicanji.json` の読込・保存
- ブラウザ内の自動保存
- 問題データを同梱した配布用単一HTMLの生成
- クイズIDごとの学習記録管理

## 基本ワークフロー

1. `DigiCanji_Editor.html` をブラウザで開く
2. 表へ問題を入力、またはCSVを読み込む
3. タイトルとクイズIDを設定する
4. `HTML生成` を押す
5. 生成されたHTMLを配布またはGitHub Pagesへ配置する

プレイヤーにはCSV読込機能を設けていません。問題データは生成時にHTMLへ同梱されます。

## 問題データ

```csv
前部,問部,後部,解答,解説
改善を,ハカル,。,図る。,しようと考えること
,アツイ,本を読む。,厚い,物を横から見たときの幅がある
選手の,コウタイ,を知らせる。,交代,人の役割などを入れ替えること
```

解答末尾と後部冒頭の句読点が同じ場合、その句読点は手書き対象から自動除外されます。

## キーボード操作（プレイヤー）

### 横書き

- `← / →`：文字枠移動
- `↑`：見直し
- `↓`：進む／回答

### 縦書き

- `↑ / ↓`：文字枠移動
- 右利き：`←` が進む／回答、`→` が見直し
- 左利き：`→` が進む／回答、`←` が見直し

### 共通

- `1～9 / 0`：1～10問目へ移動
- `Z`：1画消す
- `Esc`：メニューやオーバーレイを閉じる

## 確認項目

- JavaScript構文検査
- Chromiumでの横書き・縦書きポータブル表示
- 回転キャンバス下部への描画位置追従
- 見直し一覧の番号・問題文・入力状態配置
- 縦書きリザルトの句点削除と未回答傍線
